¿Por qué da error "Failed to fetch"?

SharePoint bloquea las peticiones que vienen de archivos locales (file://) por seguridad (CORS). Para usar este buscador tienes 3 opciones:

  • Ve a tu SharePoint → abre la biblioteca Documentos (o crea una carpeta llamada "Herramientas")

  • Sube el archivo buscador_fotos_sharepoint.html a esa biblioteca

  • Haz clic en el archivo → selecciona Abrir en el navegador (no "Descargar")

  • Al abrirse desde decoracionesamarillas.sharepoint.com, el CORS ya no bloquea nada y funciona automáticamente sin token

✅ Esta es la opción más fácil y recomendada. Una vez subido, puedes compartir el link con tu equipo y todos podrán usarlo directamente.

  • Abre tu SharePoint en Edge o Chrome y entra a tu lista de Fotos

  • Presiona F12 para abrir las herramientas de desarrollador

  • Ve a la pestaña "Consola" y pega el código que aparece en la pestaña "Obtener token"

  • Copia el token que aparece y pégalo en el campo "Token Bearer" del buscador

  • El token dura ~1 hora. Después hay que renovarlo.

Si quieres una solución permanente sin tokens, podemos crear un flujo de Power Automate que exporte los datos de tu lista a un JSON en SharePoint cada X horas, y el buscador lee ese JSON estático. Sin autenticación, sin CORS. ¿Lo hacemos?

Obtener token de acceso desde el navegador

Abre tu SharePoint en Edge o Chrome, presiona F12, ve a la pestaña Consola y pega este código: 

// Pega esto en la Consola (F12) de tu SharePoint:
fetch("/_api/contextinfo", {
  method: "POST",
  headers: { "Accept": "application/json;odata=verbose" },
  credentials: "include"
})
.then(r => r.json())
.then(d => {
  console.log("✅ Form Digest (úsalo como token):");
  console.log(d.d.GetContextWebInformation.FormDigestValue);
})
.catch(e => console.error("Error:", e));
⚠️ Importante: Este código solo funciona si lo ejecutas estando dentro de decoracionesamarillas.sharepoint.com. El valor que aparece en la consola es el token — cópialo y pégalo en el campo "Token Bearer" del buscador.

El token expira en ~1 hora. Para uso recurrente, usa la Opción 1 (subir a SharePoint).
